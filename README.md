========================================
  Yaoi - Gym Tracker
  Guia de Instalacao e Uso
========================================

REQUISITOS:
- Node.js 18+ (https://nodejs.org - versao LTS)
- Expo Go no celular (App Store / Play Store)

PRIMEIRO USO (no PC novo):
1. Instale o Node.js
2. Clone o repositorio: git clone https://github.com/gunz101/yaoi-app.git
3. Entre na pasta: cd yaoi-app
4. Clique duas vezes em "setup.bat" (ou rode no terminal)
5. Aguarde a instalacao terminar

USAR EM CASA (mesma rede Wi-Fi):
1. Clique duas vezes em "iniciar.bat"
2. No celular, abra Expo Go e escaneie o QR code
3. Pronto!

USAR FORA DE CASA (academia, 4G):
1. Clique duas vezes em "iniciar-tunnel.bat"
2. No celular, abra Expo Go - o projeto aparece na lista
3. Hosteie no pc
IMAGENS DOS EXERCICIOS:
- Na primeira vez, abra o app e va em Perfil > Configuracoes > "Baixar todas as imagens"
- Isso salva as imagens no celular (funciona offline depois)
- Precisa estar na mesma rede Wi-Fi do PC para baixar

DICAS:
- Nao desinstale o Expo Go (perde os dados do app)
- O PC precisa estar ligado para o app funcionar
- Se mudar de rede Wi-Fi, use o modo tunnel (iniciar-tunnel.bat)
